Questions fréquentes
Scribe — outil gratuit pour créer votre lettre de motivation au format ISO lettre française. Aucune donnée collectée.
Comment ça marche en 5 étapes ?
1. Collez le texte de l'offre (étape 1)
2. Renseignez vos coordonnées (étape 2)
3. Cliquez "Safe" ou "Audacieux" pour générer le prompt
4. Ouvrez ChatGPT, collez le prompt, copiez la réponse
5. Revenez sur Scribe, collez la réponse → bouton PDF disponible
Qu'est-ce qu'un "prompt" ?
Un prompt est une instruction détaillée envoyée à ChatGPT. Scribe le construit automatiquement à partir de vos informations. Vous n'avez qu'à le coller dans ChatGPT.
Pas de compte ChatGPT — c'est possible ?
Oui ! Rendez-vous sur chatgpt.com — utilisable gratuitement sans créer de compte.
Comment coller la réponse de ChatGPT ?
Dans ChatGPT : cliquez au début → Maj+clic à la fin → Ctrl+C (⌘+C Mac).
Sur Scribe : cliquez dans la zone → Ctrl+V (⌘+V Mac).
Le bouton PDF s'affiche automatiquement dès que le texte est collé.
La lettre n'est pas parfaite — que faire ?
Utilisez les boutons "§1", "§2", "§3" pour régénérer chaque paragraphe. Ou modifiez directement le texte dans la zone de réponse avant de générer le PDF.
Qu'est-ce que la formule "Monsieur le Maire" ?
Scribe détecte automatiquement les offres de mairie ou collectivité. La lettre est adressée au Maire avec la formule institutionnelle adaptée.
Mes données sont-elles confidentielles ?
Tout reste dans votre navigateur. Aucun serveur, aucune collecte, conformité RGPD totale. "Tout effacer" supprime tout en un clic.
